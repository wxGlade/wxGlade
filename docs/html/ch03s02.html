Tree Window

The tree window show the logical ierarchy of widgets and its child-widgets. For example you can see a panel as a tree's node and the widgets on it as a child nodes.

Figure 3.2. The Tree Window

Usually a frame or a panel contains a sizer, so you often see a sort of panel-sizer-widgets structure. The tree get more complex while you nest sizers on sizers.

You have a visual presentation of your widget tree that you can navigate by mouse, expand and collapse sizers, copy, cut or remove widgets.

A click on the tree window, on an icon you display properties of the corresponding element in the properties window. A double click in a frame, dialog on panel icon, make the designer window to show the corresponding Clicking with the right button of the mouse you get a pop up menu.

Figure 3.3. The menu for a widget

Figure 3.4. The menu for a sizer

The pop up menu for a widget allow you to copy, cut or remove the element. The pop up menu for a sizer allow you to copy, cut or remove the element and add or insert an empty slot as well.

Installation and requirements

While Python is a pseudo-interpreted language, you don't need any “compile” or “make” steps.

wxGlade requires Python version 2.2 or later and wxPython version 2.3.2.1 or later.

The binary versions are stand-alone and doesn't have any requirement.

Introduction

wxGlade help you to design the menu and the toolbar for your application.

You can create menu and toolbar as stand alone classes by clicking the corresponding button in the main window.

You can also create menu, toolbar and statusbar associated to a wxFrame, using relative checkboxes int the wxFrame properties window.
